# fbvideodownloadfx
Facebook Video downloader from a Video Link. This project utilizes JavaFX 8 to for One page Desktop Application look and feel.
This is a utility Desktop application to download Facebook Videos to your local Computer. Since it is a JavaFX Single page application you need to have java installed on your system. This project uses jsoup library to create a post request, capture and parse HTML file. If you are configuring this project to your local system, Please refer jsoup library from libs folder.
